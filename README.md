# Auraspace

Auraspace is a mood-based audio-visual experience app that lets users choose how they feel and instantly switch into an immersive sound and video scene.

## Features

- Mood-based home screen
- Immersive background video
- Matching music playback
- Play and pause controls
- Responsive desktop and mobile layout

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
